Step 1: Preparation is Everything

Do not skip this page. Rushing an OS installation leads to data loss.

1. Back Up Your Data

CRITICAL: Installing an OS will completely format (erase) your hard drive.

Move all your photos, documents, game saves, and important files to an external USB hard drive or a cloud provider (like Google Drive or Nextcloud). Your PC will soon be a blank slate.

2. Windows Housekeeping (If Dual-Booting)

If you plan to keep Windows alongside Linux, boot into Windows one last time:

  • Open Control Panel -> Power Options -> "Choose what the power buttons do". Turn off Fast Startup. (Fast startup locks the Windows partition, preventing Linux from reading your files).
  • If you use BitLocker encryption, suspend it, or make sure you have your recovery key printed out.

3. Create the Bootable USB

  1. Get a USB Drive (At least 8GB). Everything currently on the USB will be deleted.
  2. Download the .iso file of your chosen Linux distro (e.g., from linuxmint.com).
  3. Download a flashing tool. We highly recommend Rufus (if currently on Windows) or BalenaEtcher.
  4. Open the tool, select your USB drive, select the .iso file, and hit "Start/Flash". Wait 5-10 minutes.

Step 2: The Installation

Booting into the Live Environment

Unlike Windows, Linux lets you test-drive the OS entirely from the USB stick without touching your hard drive. This is called a "Live USB".

  1. Plug the flashed USB into your PC.
  2. Restart the PC and immediately tap your BIOS/Boot Menu key rapidly (Usually F12, F8, F2, or DEL depending on your laptop/motherboard manufacturer).
  3. From the menu, select your USB drive.
  4. If prompted by a GRUB menu, select "Try or Install Linux".
Secure Boot Issue? If your PC refuses to boot the USB, restart, enter your BIOS settings, and temporarily disable "Secure Boot". You can usually turn it back on after installation.

The Pre-Flight Check

You are now looking at the Linux desktop running from the USB. Before clicking the Install icon, verify your hardware:

  • Click the network icon. Can you connect to your Wi-Fi?
  • Open the web browser and play a video. Is the audio working?
  • Is the screen resolution correct?
  • Does your trackpad work?

If everything works, double-click the "Install" icon on the desktop. The wizard will ask for your language, keyboard layout, and timezone. When asked about partitioning, select "Erase disk and install Linux" (unless you are Dual Booting, see that tab).

Step 3: The Post-Install Checklist

Congratulations, you are running Linux! Your system is fresh, secure, and fast. Here are the first 3 things you must do to ensure a perfect experience.

1. Update the System

Open the "Software Manager" or "App Center" and click the update button. Alternatively, to feel like a hacker immediately, open the Terminal and type:

sudo apt update && sudo apt upgrade -y

This updates the core OS and all your installed apps at the same time. No forced reboots required.

2. Install Media Codecs

Because of patent laws in some countries, some Linux distros cannot pre-install the software needed to play certain `.mp4` or DVD files. In Linux Mint, simply search your app menu for "Install Multimedia Codecs" and click it.

3. Set up Timeshift (System Restore)

Linux gives you total power, which means you can break things. Timeshift is the Linux equivalent of Windows System Restore, but much better. It takes a "snapshot" of your system files.

  • Open your software manager and install `timeshift`.
  • Run it, select "RSYNC", and tell it to create a daily snapshot.
  • If an update ever breaks your PC, you can roll back to yesterday's snapshot in 2 minutes.

The File System: Where is C:?

One of the biggest hurdles for Windows users is understanding where files go. Linux does not use drive letters like C:\ or D:\.

The Root Tree ( / )

In Linux, everything starts at the Root, represented by a single forward slash: /. Every hard drive, USB stick, and folder branches out from this root.

Windows PathLinux EquivalentWhat is it?
C:\Users\YourName\/home/yourname/Your personal kingdom. Your Documents, Downloads, and app settings live here.
C:\Program Files\/usr/bin/ & /opt/Where the system installs software. You rarely need to go here manually.
C:\Windows\/etc/ & /bin/Core system files and configurations. Do not delete things here!
D:\ (USB Drive)/media/yourname/ or /mnt/External drives are "mounted" into these folders automatically when plugged in.
Pro-Tip: Because your personal files are strictly separated in `/home`, many users put `/home` on a completely separate hard drive partition. If the OS ever breaks, they can reinstall Linux on the root (`/`) without losing a single personal file!

Gaming on Linux

Ten years ago, Linux gaming was a joke. Today, thanks to Valve (creators of Steam), it is phenomenal. The Steam Deck is literally a handheld Linux PC.

Steam & Proton (The Magic Wand)

Valve created a compatibility layer called Proton. It translates Windows DirectX calls into Linux Vulkan calls in real-time.

  1. Install Steam from your App Center.
  2. Open Steam Settings -> Compatibility.
  3. Check the box: "Enable Steam Play for all other titles".
  4. Click Install on your Windows games. They will just run.

Check ProtonDB.com to see how well specific games run. Over 80% of the top 1000 Steam games are rated "Gold" or "Platinum".

Epic Games, GOG, and Amazon Games

Do not try to install the official Epic Games launcher. Instead, open your Linux App Center and install the Heroic Games Launcher. It is an open-source client that logs into your Epic/GOG/Amazon accounts and downloads your games, automatically applying the Proton translation layer so they run perfectly on Linux.

The Anti-Cheat Wall: The only genre that struggles on Linux is competitive multiplayer games that use kernel-level anti-cheat (e.g., Valorant, Destiny 2, Rainbow Six Siege, Fortnite). The publishers actively block Linux to prevent cheating. If you play these daily, you MUST keep a Windows Dual-Boot.

Advanced: Dual Booting (Windows + Linux)

Need Windows for Valorant or Adobe, but want Linux for privacy, coding, and daily driving? Dual booting is the answer.

The Concept

You split your hard drive into two sections (partitions). Windows lives in one, Linux in the other. When you turn on your PC, a menu called GRUB asks you which system to start.

Step-by-Step Guide

  1. In Windows: Right-click the Start button -> "Disk Management".
  2. Right-click your C: Drive and select "Shrink Volume".
  3. Enter the amount to shrink (e.g., `60000` MB for 60GB). Click Shrink. You now have "Unallocated Space".
  4. Insert your Linux USB and boot into the Live Environment.
  5. Run the Linux Installer.
  6. When asked "Installation Type", select the option: "Install Linux alongside Windows Boot Manager". The installer is smart enough to find the Unallocated Space and put Linux exactly there without touching Windows.
  7. Finish the install and reboot. You will see the GRUB boot menu!
Note on Time Desync: Windows and Linux read the hardware clock differently. After dual-booting, your Windows clock might be off by a few hours. To fix this, open the Linux terminal and run:
timedatectl set-local-rtc 1 --adjust-system-clock

Installing Software via Terminal

Instead of hunting for `.exe` files on ad-filled websites, Linux uses "Package Managers" (like an App Store for the terminal). Example for Debian/Ubuntu/Mint systems (using apt):

sudo apt update
sudo apt install vlc

Boom. VLC Media Player is downloaded from official servers and installed. To remove it: sudo apt remove vlc.

The Complete Linux Glossary

The Linux community uses a lot of jargon. Here is your translation manual:

  • Distro (Distribution): A specific flavor of Linux. It's the Linux kernel bundled with a desktop interface and pre-installed apps. (e.g., Ubuntu, Fedora, Pop!_OS).
  • Kernel: The core engine of the OS. It talks directly to your CPU, RAM, and hardware drivers.
  • DE (Desktop Environment): The visual GUI. The taskbar, the start menu, the window borders. (e.g., GNOME, KDE Plasma, XFCE). You can install multiple DEs on one Linux system!
  • Wayland: The modern display protocol that draws pixels on your screen. It handles multi-monitor HDR and different refresh rates natively, replacing the legacy X11.
  • PipeWire: The modern audio and video handling system. If your Bluetooth headphones switch seamlessly, thank PipeWire.
  • Flatpak: The modern way to package apps. It's a universal file format that works on ANY distro and is sandboxed for security.
  • Snap: Canonical's (Ubuntu's maker) alternative to Flatpak. Many users prefer Flatpak because it is completely decentralized.
  • AppImage: A portable app format. You download one file, make it executable, and double-click it to run. No installation required. (Similar to Windows Portable apps).
  • Repository (Repo): A secure, official server maintained by your Distro where all verified software is downloaded from.
  • GRUB: The Grand Unified Bootloader. The menu you see when turning on a dual-boot PC.
